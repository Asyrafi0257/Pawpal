# Pawpal Pet Adaption and Donation

#Setup

1. install the dependencies and run the command :
   flutter pub get
   
2. install Xampp
   
3. Import API folder to Xampp
   
4. Change IP Address at myconfig.dart
   
5. Run the app

#API explanation

1. dbconnect.php
   handle the database connection
   
2.login.php
  handle user login and authentication
  
3.register_user
  handle user registration and encryp password and check email already
  
4.get_my_pets.php
  to load everything about pets and load pets based on search query
  
5.submit_pets.php
  handle submissionof pets and imagesof the pets
